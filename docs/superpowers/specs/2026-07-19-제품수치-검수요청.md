세 파일(대형 F·J 시리즈, 중형 D·D4·H·M·P 시리즈, 소형 S2·DHS1·DHS2·S·T 시리즈)을 모두 합산한 전 시리즈 범위.

**⚠ 참조값과의 차이 및 사유**: 브리프에 제시된 확인 예시는 "플런저펌프 F5000/F6000 계열 — 200 bar / 1717 L/min / 530 kW"였습니다. 그러나 제가 열람한 3개 카탈로그 파일에는 `F5000`/`F6000`이라는 모델명이 존재하지 않으며, 실제 모델 체계는 대형(F/J), 중형(D/D4/H/M/P), 소형(S2/DHS1/DHS2/S/T) 시리즈로 구성되어 있습니다. 브리프 지시대로 "small/medium 시리즈까지 포함하면 범위가 넓어질 수 있다"는 점에 따라 전 시리즈를 아우르는 범위(80–560 bar / 10–1,924 L/min / 3–790 kW)로 기재했습니다. 대형 J시리즈 단독 최대치는 압력 445 bar / 유량 1,924 L/min / 출력 790 kW로, 참조값(200bar/1717L/min/530kW)과 유사한 특정 모델이 카탈로그 내에 별도로 존재할 가능성이 있어 보이나, 제공된 이미지에서는 확인되지 않았습니다. 실제 F5000/F6000 모델 카탈로그가 별도로 있다면 파일을 알려주시면 재판독하겠습니다.
